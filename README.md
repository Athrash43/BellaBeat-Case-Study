# Bellabeat Case Study

## Overview
This project analyzes Fitbit smart device activity and sleep data to identify user behavior trends and develop marketing recommendations for Bellabeat, a wellness technology company.

## Business Task
Analyze smart device usage trends and recommend how Bellabeat could use activity and sleep insights to improve marketing strategy and user engagement.

## Tools Used
- Google Sheets
- Pivot tables
- Summary tables
- Charts

## Data
The analysis used Fitbit Fitness Tracker data, focusing on:
- `dailyActivity_merged.csv`
- `sleepDay_merged.csv`

## Process
I preserved the raw data, created helper columns for clean dates and minutes awake in bed, summarized activity and sleep metrics at the user level, joined the summaries by user ID, flagged potential data quality issues, segmented users by average daily steps, and created charts to compare activity, calories, sleep, and data completeness.

## Key Findings
- The activity dataset included 35 users, and 24 users had sleep data.
- Most users were low or moderate activity users.
- Average calories burned increased with activity level.
- Sleep data was less complete than activity data.
- Low-activity users spent more time awake in bed on average.

## Recommendations
1. Personalize app messaging/fitness plans by segment for BellaBeat users.

2. Promote sleep tracking more through reminders, recovery insights, and weekly summaries.

3. Position BellaBeat subscription marketing to be a personalized recovery, activity, hydration and sleep tracker that delivers real        insights and manageable goals by activity level.
